# Banana navigation project from Udacity deep RL nanodegree.
In this project we train an agent to navigate a 2D space collecting yellow bananas and avoiding blue bananas.

## Setup
First, we need to install the necessary Python packages and set up the environment. Follow steps 1-6. from [this](https://github.com/udacity/deep-reinforcement-learning#dependencies) github page.[^1]

[^1]: During step 3., install the required environments by running:
```
pip install gym[classic_control]
pip install gym[box2d]
```
